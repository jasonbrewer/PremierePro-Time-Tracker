How it works

Set up once. Works forever.

No ongoing configuration. No daily habits to build.

01
Add your projects

Click "Add Project", give it a name, and link it to your .prproj or Premiere Production file. Premiere Tracker uses macOS security bookmarks — it remembers the file even if you move it.

02
Open Premiere and work

That's it. Premiere Tracker detects when Premiere Pro is running and which project file is open, then starts the timer automatically.

03
Check your time anytime

Open Premiere Tracker to see a live timer for the current session, total time per project, and a daily breakdown — all in one clean window.

Privacy Policy

Last updated: March 2025

Overview

Premiere Tracker is designed with privacy as a core principle. The app does not collect, transmit, or share any personal data.

Data We Collect

We collect no data. Premiere Tracker does not connect to the internet. All information — project names, file paths, and time logs — is stored exclusively on your Mac using local storage (UserDefaults).

What the App Stores Locally

  • Project names you enter
  • File paths and security bookmarks to your linked .prproj or .prodset files
  • Time tracking data (session durations and daily logs)

This data never leaves your device and can be deleted at any time by removing the app.

Third-Party Services

Premiere Tracker uses no third-party SDKs, analytics tools, crash reporting services, or advertising frameworks.

File System Access

Premiere Tracker requests access to specific files and folders you choose when linking a project. This access is used solely to detect whether a project is currently open in Adobe Premiere Pro. The app reads file metadata only and does not read, modify, or transmit the contents of your project files.

Children's Privacy

Premiere Tracker does not collect any data from any users, including children under the age of 13.

Changes to This Policy

If this policy changes, the updated version will be posted on this page with a revised date. As the app collects no data, no material changes to data handling are anticipated.

Contact

Questions about this policy? Reach out via the Mac App Store support link on the app's product page.
